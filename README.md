# Проект парсинга json-данных с Eshop

### Пакеты

- __Модуль "JSON_Extractor" - содержит функции, выгружающие данные с сайта Nintendo__
- __Модуль "SQLAlchemy_Core" - содержит описание структуры базы данных__

### Скрипты
- __loadJson.py - выгружает данные продукта и сохраняет в json__
- __readJson.py - переносит данные в базу данных (данные берутся из первого найденного json-файла каталога `readable_json`)__
- __getImagesFromBinary - позволяет выгрузить изображение обложки товара из базы данных (данные сохраняются в директорию `retrieved_images`)__

Для примера в проекте присутствует база данных и выгруженные json-файлы

Для обновления зависимостей:
``pip freeze > requirements.txt``

Для установки зависимостей:
``pip install -r requirements.txt``
